## 📖 О проекте

**Aurora Dentistry** — современный адаптивный лендинг, разработанный с нуля без использования готовых шаблонов и конструкторов.

Основная цель проекта — создать стильный, легкий и вызывающий доверие интерфейс, который подчеркнет высокий уровень стоматологической клиники и обеспечит удобство взаимодействия для пользователей на любых устройствах.

## 💻 Запуск проекта

1. Клонируйте репозиторий

```bash
git clone https://github.com/Issedafuu/aurora-dentistry.git
```

2. Перейдите в папку проекта

```bash
cd aurora-dentistry
```

3. Откройте `index.html` в браузере

или воспользуйтесь расширением **Live Server** в Visual Studio Code.
