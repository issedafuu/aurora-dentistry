## ✨ О проекте

**Aurora** — это стильный, атмосферный и продающий одностраничный сайт для частной стоматологической клиники.  

Проект разработан **с нуля**, без использования готовых макетов. Главная цель — создать ощущение доверия, комфорта и высокого профессионализма. Дизайн получился лёгким, современным и премиальным, идеально подходящим для медицинской тематики.

### 🎯 Что внутри
*   **Полностью адаптивная вёрстка** — идеально смотрится на телефонах, планшетах и десктопах.
*   **Живые анимации** — плавное появление блоков при скролле, анимированные градиентные блобы в шапке.
*   **Сложные UI-элементы** — многослойные карточки услуг с эффектами стекла и hover-анимациями.
*   **Модальные окна** — удобная запись на приём и быстрый просмотр цен без перезагрузки страницы.
*   **Микроанимации** — мерцающие звёздочки в логотипе и акцентных элементах.
*   **Preloader** — фирменная анимация загрузки в стиле Aurora.
*   **SEO-оптимизация** — семантическая вёрстка, мета-теги, заголовки.
