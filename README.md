# Стоматология Aurora

**Современный премиальный лендинг для стоматологической клиники в Екатеринбурге**

## О проекте

**Aurora** — это стильный, атмосферный и продающий одностраничный сайт для частной стоматологической клиники. 

Проект разработан **с нуля**, без использования готовых макетов. Главная цель — создать ощущение доверия, комфорта и высокого профессионализма. Дизайн получился лёгким, современным и премиальным, идеально подходящим для медицинской тематики.

## ✨ Ключевые особенности

- Чистый и современный дизайн в голубых тонах
- Анимированные градиентные блобы в шапке сайта
- Сложные многослойные карточки с hover-эффектами
- Полностью адаптивная вёрстка (мобильные устройства, планшеты, десктоп)
- Плавные анимации появления блоков при скролле
- Удобные модальные окна (запись на приём и просмотр прайса)
- Красивое уведомление об успешной отправке формы
- Preloader с фирменным стилем
- Оптимизация под поисковые системы (SEO)

## Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — CSS Variables, Grid, Flexbox, `clamp()`, keyframes, backdrop-filter
- **Vanilla JavaScript** — чистый JS без фреймворков
- Адаптивная типографика и современные CSS-техники
